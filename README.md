# mydotfiles
My thinkpad dotfiles. Work on gentoo linux, archlinux and maybe on linux from scratch with some changes

## OS
 - Archlinux
 - Gentoo
 - Lfs (maybe)

## Soft
- Will be automaticly downloaded.

## How to use?
- Go to folder with dots on your system
- Run `chmod +x apply && ./apply`
- Wait.
